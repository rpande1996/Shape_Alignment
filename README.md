## Overview

This code aligns image 2 in the shape of image 1.

## Softwares

* Recommended IDE: PyCharm 2021.2

## Libraries

* Numpy 1.21.2
* OpenCV 4.6.0
* Matplotlib 3.5.0
* SciPy 1.9.1

## Programming Languages

* Python 3.8.12
